Magic Match

Magic Match is a runic memory game, where you have to make pairs of runes before the timer runs down. But if one of the runes is colored it activates an special effect depending on the rune!

  • Fire rune: Instantly makes a pair
  • Wind rune: Temporarely flips some runes
  • Ice rune: Freezes the timer for a bit
  • Death rune: Instantly loses the game
